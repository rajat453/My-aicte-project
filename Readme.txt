//Secure Data Hiding in Images Using Steganography//

**Introduction**

This project implements steganography to securely hide data within images, ensuring confidentiality and protection from unauthorized access.

**Features**

Hide secret data inside image files

Maintain image quality while embedding data

Extract hidden data securely

Optional encryption for enhanced security

**Requirements**

Python 3.x

Libraries: OpenCV, NumPy, Pillow

**Installation**
pip install opencv-python

**Usage**
Hide Data: Encode a secret message into an image.
Extract Data: Retrieve the hidden message from the image.
